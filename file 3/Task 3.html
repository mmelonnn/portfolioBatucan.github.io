Task 3: Style a List

  • This list has a marker color of green
  • The list items have background colors of light blue
  • Each list item has a 5-pixel margin at the top and bottom
  • The list items have font sizes of 20 pixels
  • This list has a padding of 10 pixels
ul.a {
        list-style:none;
        background-color: lightblue;
        padding: 10px;
        font-size: 20px;
        margin:4px;
    }
    
    ul li::before {
        content: "\2022";
        color: green;
        font-weight: bold;
        display: inline-block; 
        width: 1em;
        margin-left: 1em;
      }
    
    li.a {
        margin-top: 5px;
        margin-bottom: 5px;
        color:green;
    }
